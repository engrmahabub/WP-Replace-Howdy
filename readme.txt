== Description ==
A wordpress plugin for replace Replace Howdy with another word.

= Features =
* Replace Howdy with another word.

== Installation ==
1. Upload the entire `replace-howdy-with-other-word` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now Replace Howdy is prepared to use.

= Usage =

1. Go to Settings > SM Replace Howdy.
2. Write a word on text box.
3. Hit the save button.
4. Refresh your browser.
